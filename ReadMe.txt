// LIKE

. The LIKE function allows you to automatically like articles on a Vinted page.

- STEP 1 -
Define the number of clicks in parameter.

- STEP 2 -
Define the time interval between each click.

- STEP 3 -
Save the changes.

- STEP 4 -
Close and reopen Chrome to apply changes.

- STEP 5 -
Open the Vinted home page or a Vinted search page.

- STEP 6 -
Open VBot, click on LIKE and wait.

- STEP 7 -
VBot is launched !
